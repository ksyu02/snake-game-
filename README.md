# 🐍 Snake Game with Database

Проект классической игры Snake с сохранением рекордов игроков в базе данных SQLite и серверной частью на Node.js.

## 📌 Описание проекта

Игрок управляет змейкой, собирает еду и набирает очки.  
После завершения игры результат сохраняется в базе данных через серверное API.



## ⚙️ Используемые технологии

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- SQLite3
- CORS

## 🗄️ База данных

В проекте используется SQLite база данных.

Создана таблица scores:

```sql
CREATE TABLE IF NOT EXISTS scores (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    player_name TEXT NOT NULL,
    score INTEGER NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
📌 Описание полей:
id — уникальный идентификатор записи
player_name — имя игрока
score — набранные очки
created_at — дата и время записи


🚀 Запуск проекта
1. Установка зависимостей

npm install

2. Запуск сервера

node server.js

3. Открыть игру в браузере

http://localhost:3000

📡 API

Получить рекорды

GET /scores

Сохранить результат игры

POST /scores
